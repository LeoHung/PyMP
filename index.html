LazyMP

Make your python script running in parallel with few annotations.

Less Code

You do not need to rewrite your Python code into complex C extention. Just add "#pragma omp parallel for" to make your loop running in parallel!

Run Fast

LazyMP can make your for-loop in N/2 ~ Nx speedup. Much better than using Python threads.

Easy to Use

Adding the annotation does not break your code. If your machine does not support LazyMP, the code still can be executed.
